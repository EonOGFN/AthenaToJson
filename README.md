### Simple Tool for Fortnite Modding Communities
- **Convert Item IDs to JSON for Private Servers with ease.**
---
## Format Output Example
```json
"AthenaCharacter:Example": {
  "templateId": "AthenaCharacter:Example",
  "attributes": {
    "item_seen": false,
    "variants": [],
    "favorite": false
  },
  "quantity": 1
}
```
